# DragosEngine

DragosEngine is a C++ core game engine with .NET (C#) scripting, DirectX12 rendering, PhysX + Box2D physics, ECS architecture and a Qt-based editor.

## Quick start
1. Install prerequisites: Visual Studio 2022/2026, CMake >= 3.20, DirectX SDK, Qt dev kit, .NET runtime.
2. Configure and build:
   mkdir build && cd build
   cmake .. -G "Visual Studio 17 2022" -A x64
   cmake --build . --config Release
